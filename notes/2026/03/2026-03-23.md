## 핵심 개념

### Web 3의 역사
- 1960년대, 미국 국방부 산하 고등연구계획국(ARPA)은 핵 공격 이후에도 통신을 유지할 수 있도록 설계된 분산 네트워크 시스템인 ARPANET을 개발
- Web 1.0 시대는 1990년 대 초 월드 와이드 웹(WWW)의 등장과 함꼐 시작
- Web 2.0 시대는 web 1.0 문제점을 보완하기 위해 등장, 2004년경 상호작용적인 사회적 웹사이트로의 전환 (blog, forum, tiktok 등)
- WEb 3.0 시대에는 web 2.0시대 데이터 소유권 문제, 플랫폼 독점 문제를 보완하기 위한 분산형 사용자 주도 네트워크 구축 및 탈중앙화를 강조한 체재 구체화

결론: 암호화 기술의 선구자인 비탈릭 부태린 등이 2015년 에 출시한 오픈소스 탈중앙화 블록체인 플랫폼 '이더리움'과 스마트 컨트랙트라는 중요한 혁신을 도입하며 DApp 코드를 신뢰할 수 있는 제 3자에 의존하지 않고 전 세계적으로 실행할 수 있도록 함

### Web3 신흥 기술
- 대체 불가능 토큰 (NFT): 스마트 계약으로 보호되어 고유성과 희소성을 보장하는 가치형 암호화 자산
- 분산형 자율 조직(DAO): 자금 배분, 규칙 설정, 프로젝트 개발 방향과 같은 중요한 사항에 대해서는 조직 토큰을 보유하고 집단 투표를 통해 결정하는 구성원들이 참여
- 사회적 금융(SocialFi): 사용자가 게시물 작성, 댓글 달기, 콘텐츠 공유와 같은 소셜 활동에 참여할 경우 토큰이나 NFT와 같은 실질적인 경제적 보상을 제공
- 게임 파이낸스(GameFi): NFT을 통합한 게임 모델 -> 가상 게임 세계에 참여 가능
- 분산형 신원(DID): 사용자가 복잡한 이더리움 지갑 주소를 읽기 쉽고 기억하기 쉬운 도메인 이름으로 변환

작업증명(PoW)방식은 높은 에너지 소비량으로 비판받아 온 반면, 환경 친화적인 지분증명(PoS) 방식이 점차 선호

작업증명(Proof of Work): 일을 많이 한 사람에게 권한을 준다. (채굴자에게 혜택을 더 준다)
지분증명(Proof of Stake): 돈(자산)을 예치한 사람에게 권한을 준다(주주 총회에 주식을 많이 가진 의결권과 비슷)
