# Recipe Contest (2026-06-11)

## Deploy (배포 환경 구축)

- Hardhat Ignition을 활용한 RecipeContest.sol 스마트 컨트랙트 배포 구조 분석 및 적용 준비
- 기본 스크립트 배포 방식에서 고도화된 모듈 기반 배포로 파이프라인 전환
- ignition/modules/ 디렉토리 내 사용자 정의 컨트랙트 배포 파일 매핑 구조 확인

## Log / Trouble
- 문제 발생: npx hardhat ignition deploy ignition/modules/Lock.ts --network sepolia 명령어 실행 시 Could not find a module file 에러 출력
- 원인 분석: * 튜토리얼 예제 파일인 Lock.ts를 타겟으로 지정했으나, 현재 프로젝트 내에는 존재하지 않음 (기본 예제로 Counter.ts만 존재함).
- 해결: RecipeContest.sol은 컴파일된 컨트랙트 파일일 뿐이며, 이를 Ignition으로 배포하기 위해서는 buildModule을 포함하는 별도의 .ts 배포 모듈 파일이 필요함을 확인.
