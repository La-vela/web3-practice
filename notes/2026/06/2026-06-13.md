# Bicycle Collection (2026-06-13)
배달 자전거 도감 스마트 컨트랙트

## Features

### Bicycle Collection 스마트 컨트랙트 초안 작성
- `ERC-1155` 기반 자전거 도감 컨트랙트 구조 설계
- 관리자 전용 아이템, 게임 운영 전용 아이템, 해금(민팅) 아이템 추가
- 해금 여부 조회, 아이템 등록 여부 메타데이터 조회 구조
- Hardhat Ignition용 배포 모듈 초안
- Hardhat 프로젝트 초기화
- OpenZeppelin 설치 후 `BicycleCollection.sol` 컴파일
- 로컬 테스트 함수 작성
- Sepolia 테스트넷 배포
- React + MetaMask로 민팅 테스트 페이지 연결
- Unity 게임에서는 NFT 조회 결과만 반영하는 방식 검토
