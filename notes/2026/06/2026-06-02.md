## Log / Trouble

### 이슈(Issue): ERR_MODULE_NOT_FOUND (v3 모듈 호환성)
- 원인: Hardhat v3의 엄격한 ESM 모듈 방식과 'ethers' 플러그인 간의 참조 출동.
- 해결: 'import hre from "harhat"; const { ethers } = hre;'

### Issue: HHE902 의존성 누락
- 원인: 프로젝트 이동 및 초기화 과정에서 @openzepplin/contracts 라이브러리 미설치
- 해결: 'npm install @openzeppelin/contraccts' 명령어로 필수 라이브러리 보강 후 컴파일 성공.
