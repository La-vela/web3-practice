# Recipe Contest (2026-06-08)

## Local Deployment & Function Call Test

### 1. Hardhat Node 실행

```bash
npx hardhat node
```

### 2. 컨트랙트 배포

```bash
npx hardhat run scripts/deploy.ts --network localhost
```

결과:

```text
RecipeContest deployed to:
0x5FbDB2315678afecb367f032d93F642f64180aa3
```

### 3. Hardhat Console 접속

```bash
npx hardhat console --network localhost
```

### 4. 컨트랙트 연결

```javascript
const RecipeContest = await ethers.getContractFactory("RecipeContest");

const contract = await RecipeContest.attach(
  "0x5FbDB2315678afecb367f032d93F642f64180aa3"
);
```

### 5. 레시피 제출

```javascript
await contract.submitRecipe(
  "Wasabi Burger",
  "ipfs://wasabi-burger"
);
```

확인:

```javascript
await contract.recipes(1);
```

상태:

```text
0 = PENDING
```

### 6. 관리자 승인

```javascript
await contract.approveRecipe(1);
```

확인:

```javascript
await contract.recipes(1);
```

상태:

```text
1 = APPROVED
```

### 7. NFT 발행

```javascript
await contract.mintRecipe(1, 1);
```

확인:

```javascript
await contract.recipes(1);
```

상태:

```text
3 = MINTED
```

### 8. NFT 보유량 확인

```javascript
const [owner] = await ethers.getSigners();

await contract.balanceOf(owner.address, 1);
```

결과:

```text
1n
```
