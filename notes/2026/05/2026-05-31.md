## Learn

### ERC1155 - batch, balanceOf

이것이 ERC1155의 장점, 아이템 10종류를 한꺼번에 획득할 때 ERC721은 10번 전송해야 하지만 ERC1155는 단 1번으로 끝남
- 비용 절약 측면에서 좋음

- _mintBatch로 여러 종류의 아이템을 한 트랜잭션으로 보낼 수 있음

예시1-1)
```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

// 상속
import "@openzepplin/contracts/token/ERC1155/ERC1155.sol";

function rewardBundle(address creator) public {
    uint256[] memory ids = new uint256[](2);
    ids[0] = LEGENDARY_RECIPE;
    ids[1] = COOK_BADGE;

    uint256[] memory amounts = new uint256[](2);
    amounts[0] = 1;  // NFT 1개
    amounts[1] = 10; // 배지 10개

    _mintBatch(creator, ids, amounts, "");  // Batch 기능(한 트랜잭션)
}

```

예시1-2)외부 컨트랙트
- balanceOf(account,id)이 조회 함수로 '조건문'이 됨
```solidity
contract RecipeContest {
    RecipeGame public recipeContract;

    constructor(address _recipeAddr) {
        recipeContract = RecipeGame(_recipeAddr);
    }

    function calculateBonus(address player) public view returns (uint256) {
        // 플레이어가 가진 COOK_BADGE(ID: 2) 개수를 확인
        uint256 badgeCount = recipeContract.balanceOf(player, 2);
        
        // 배지 1개당 스코어 100점씩 가산
        return badgeCount * 100;
    }
}
```
