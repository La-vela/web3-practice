### contract 생성자 정의
- pragma 컴파일러 버전 정의
- contract 생성
- constructor 정의
- 각 주소의 해당 토큰 잔액을 기록할 데이터 구조를 정의
    - _balances 잔액
    - _totalSupply 동전의 총량
    - _owner 계약 소유자 및 코인 발행 관리자
- 토큰 잔액을 효율적으로 추적하기 위해 **매핑 데이터** 구조를 사용 (mapping)
- 이제 각 주소의 잔액을 나타내는 잔액 매핑 (totalSupply)
- 관리자 정의 - 신원을 나타내는 주소 변수 사용 (address private owner)
    - 계약 에서 배포자를 확인하는 유일한 방법은 생성자 에서 msg.sender 구문을 사용

```
// SPDX-License-Identifier: MIT

pragma solidity 0.8.17;
contract MyToken {

    // mapping used to store the balance corresponding to each address
    mapping(address => uint256) private balances;

    //A uint256 variable is used to store the total supply of the token. It is defined as public and can be queried by anyone.
    uint256 public totalSupply;

    //An address variable is used to store the issuer of this token. This is used for some permission control
    address private owner;

    constructor() {
        owner = msg.sender;
    }

    
}

```
