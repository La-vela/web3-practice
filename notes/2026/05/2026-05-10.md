## Learn


### receive (수신 함수)
누군가가 추가 데이터 없이 이더를 계약으로 보낼 때 실행되는 특정 함수
```
// Define an empty receive function.
receive() external payable { }
```

### fallback 
존재하지 않는 함수를 호출하려 하거나, 이더는 전송되었지만 수신 기능이 존재하지 않을 때, 수신 기능까지 존재하지만 메시지 데이터가 비어있을 때
즉, 계약에 수신 함수가 정의되어 있지 않고 대체 함수 또한 지불이 불가능한 경우 실행되는 fallback 함수
```
fallback() external payable { }
```

### selfdestruct (자폭? 함수)
계약을 중지하고 , 남은 이더를 지정된 주소 로 전송하며 , 향후 상호 작용을 위해 계약을 비활성화하는 내장 함수 
```
// Write a selfdestruct statement, and set its target address to be target.
selfdestruct(target);
```


## example
```
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

contract ReceiveFallbackSelfdestruct {

    address public owner;

    // 배포자 저장
    constructor() {
        owner = msg.sender;
    }

    // 데이터 없이 ETH만 보냈을 때 실행
    receive() external payable {
        emit Received(msg.sender, msg.value);
    }

    // 존재하지 않는 함수 호출 or 데이터 포함 송금 시 실행
    fallback() external payable {
        emit FallbackCalled(msg.sender, msg.value, msg.data);
    }

    // 잔액 조회
    function getBalance() public view returns(uint) {
        return address(this).balance;
    }

    // 계약 삭제 + 남은 ETH 전송
    function destroy(address payable target) public {

        require(msg.sender == owner, "Only owner");
        emit Destroyed(target, address(this).balance);

        selfdestruct(target);
    }
}
```
