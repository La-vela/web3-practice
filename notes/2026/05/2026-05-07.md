### inheritance (상속)

상속 코드 (is) 예시
```
pragma solidity ^0.8.0;

contract ParentContract {
    // State variables and functions in the parent contract.
    uint public parentVariable = 1;


    function parentFunction() public pure returns (string memory)   {
        return "Helllo from parent contract!";
    }
}

contract ChildContract is ParentContract {
    // state variables and functions in the child contract.
    uint public childVariable;

    function childFunctionCallParentFunction() public pure returns (string memory) {
        return parentFunction();
    }

    function childFunctionUsingParentVariable() public {
        childVariable = parentVariable;
    }
}
```

ERC20 컨트랙트 상속도 가능
```
contract examtoken is ERC20 {}
```

상속받은 계약의 생성자를 초기화
- 계약 A가 계약 B를 상속받을 떄, B의 코드가 A로 복사
- 올바른 초기화를 위해, A는 자체 생성자 내에서 B의 생성자를 호출한 후 생성 과정을 진행
```
// contract B
contract B {
    uint public bValue;

    constructor(uint _value) {
        bValue = _value;
    }
}

// conract A inherit contract B
contract A is B {
    uint public aValue;

    constructor(uint _valueA, uint _valueB) B(_valueB) {
        aValue = _valueA;
    }
}
```
