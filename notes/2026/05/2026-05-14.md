## Learn


### low level call (저수준 호출)
- 함수 호출 하나만 할 경우 전체 계약 내용을 파일에 정의하지 않으면서 사용할 때 유용하다.
- 예를들어 NFT 계약을 통해 NFT를 구매하고 싶은데, 단순히 NFT를 구매하기 위해 IERC721 프로토콜과 NFT 계약을 직접 정의하고 싶지 않고, 간단한 호출만으로 해결이 가능하다.

Call
```
// 특정 주소로 이더를 보내거나, 다른 컨트랙트의 함수를 실행하는 방식
(bool success, bytes memory data) = address(targetAddress).call{value: amount}(abiEncodedData); // 이더와 함께 호출 함수 인코딩
```

delegatecall
```
// 호출당한 컨트랙트의 코드를 가져와 실행하는 방식
address(targetAddress).delegatecall(abi.encodeWithSignature("dosome(uint256)", 5));
```
