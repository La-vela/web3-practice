# Web3 / 블록체인 기초

## [Web의 진화]
### Web 1.0 (1990년대 ~ 2004년경)
- 읽기 중심(Read-only)
- 정적 웹페이지
- 사용자 참여 제한적
ex) yahoo, 개인 html 홈페이지

### Web 2.0 (2004년 ~ 현재)
- 읽기 + 쓰기(read & write)
- SNS, 플랫폼 중심
- 사용자 생성 콘텐츠 활성화
- 데이터가 기업 서버에 집중
ex) YouTube, Facebook, Instagram

### Web 3.0 (2014년 ~ 현재/미래)
- 읽기 + 쓰기 + 소유(Read, Write, Own)
- 탈중앙화
- 블록체인 기반
- 사용자 데이터 / 자산 소유권 가조
- 스마트 컨트랙트 활용
ex) Ethereum, NFT, DAO, DeFi 서비스
