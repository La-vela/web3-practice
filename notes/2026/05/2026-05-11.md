### abi
abi.encode 내장함수의 집합 (전역 변수): 매개변수 목록을 바이트 배열로 변환
abi.decode 함수 호출 결과나 트랜잭션 입력 데이터와 같은 인코딩된 원시 데이터를 압축 해제하고 읽기 쉬운 데이터 구조로 변환
abi.encodepacked: 추가적인 패딩이나 구조를 넣지 않고 데이터를 촘촘하게 압축하는 방법
- 숫자를 넣으면 encode와 달리 크기만큼 차지해 해시함수(keccak256)를 쓸 때 특정 데이터를 조합해서 고유의 ID값을 만들고 싶을 떄 쓴다.
- 단, 다른 데이터 타입끼리 섞여 있을 때 의도치 않게 데이터가 겹쳐서 해시 충돌
```
bytes memory encodedData = abi.encode(param1, param2);
address decodedAddress = abi.decode(encodedData, (address));

bytes memory encodedData = abi.encodePacked(param1, param2);
```

기존 국제 송금 시스템과 현지 법정화폐 통합의 복잡성에서 비롯되는 어려움을 해결하기 위한 설계


thirdPartyController.ts: 가상화폐와 법정화폐(IDRX) 간의 환율 계산, Gemini AI 기반 챗봇 기능 담당 역할을 하는 Controller.ts

코드 중, 입력 데이터 유효성 검사(Input Validation) :
환전 과정에서  사용자의 입력 문장을 정규표현식과 시스템이 설정한 코인만 데이터로 받아서검증하는 코드
```
const parseConversionRequest = (input: string) => {

    // 1. 정규표현식(Regex) 정의
    const regex = /(\d+(?:\.\d+)?)\s*([A-Za-z]+)\s*(?:ke|to)\s*([A-Za-z]+)/i;

    // 2. 입력받은 문장(input)과 위 정규식을 대조
    const match = input.match(regex);

    // 만약 "10 ETH to IDRX" 같은 패턴이 없으면 null 반환
    if (!match) return null;
    
    // 3. 추출된 결과값을 변수에 저장
    const amount = parseFloat(match[1]); // 숫자 추출
    const fromCurrency = match[2].toUpperCase(); // 첫 번째 통화 추출
    const toCurrency = match[3].toUpperCase(); // 두 번째 통화 추출

    // 4. 안전장치: 지원하지 않는 통화는 에러 발생
    if (!supportedFromCurrencies.includes(fromCurrency) || !supportedToCurrencies.includes(toCurrency)) {
     throw new Error("Unsupported currency conversion");
    }

    // 5. 정리된 데이터 반환
    return { amount, fromCurrency, toCurrency };
};

```
