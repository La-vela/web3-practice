### Library

- 컨트랙트랑 구조와 정의 방법은 같다.
- 라이브러리는 상태 변수를 선언할 수 없다(일처리가 끝나면 증발) 

학생 라이브러리 정의 (S1, S2)
```
library StudentLibrary{
    struct Student{
        string name;
        uint id;
        uint grade;
    }

    function compareGrades(Student S1, Student S2) returns(Student){
        //function body
    }
}
```

라이브러리 호출
```
StudentLibrary.compareGrades(a, b);
```
