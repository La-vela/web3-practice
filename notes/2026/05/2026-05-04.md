## Learn

### revert

require는 검증 혹은 간단 조건 체크
revert는 복잡한 조건 검증, 가스 효율을 위해 Custom Error와 함께 사용
```
revert(); // 기존 형식
revert("error message"); // 에러 메시지 추출
revert CustomError(5, true, ...);    //데이터(인자)를 정확히 넘겨, 예를 들어 컴퓨터가 address(0x123)떄문에 실패한 것을 확인 후 자동처리 가능
```

실제 오류 유형 정의 방법
```
error unauthorizedAccess();
error insufficientFunds(uint256 balance, uint256 withdrawalAmount);

function withdraw(uint256 amount) public {
    if (msg.sender != owner) {
        revert uauthorizedAccess();
    }
    if (balance[msg.sender] < amount) {
        revert insufficientFunds(balance[msg.sender], amount);
    }
    // ... rest of the function ...
}
```

핵심은 error message만큼 글자 길이에 비례해 가스 낭비를 보완하기 위해 4바이트인 해시 데이터를 던져줘 가스비를 절약
