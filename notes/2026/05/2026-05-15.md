## Learn


### Function Modifier (함수 수정자)
_; 는 수정자가 적용된 함수의 원래 본문(Body)이 들어갈 위치: 나머지 함수 코드를 여기에서 실행해라

```
modifier onlyOwner() {
require(msg.sender == owner, "Only the owner can call this function.");
_;
}
```
- onlyOwner 수정자
- require 문 통과
- _;
