## Learn


### abstract (추상적인 계약)
추상방식 이라는 것이 내용은 부모 계약에선 비워두고, 자식 계약에서 Override를 통해 직접 구현해야한다.
```
abstract contract ContractA { }
```

### keccak256 (전역 함수)
Solidity가 강타입 언어이기도 하지만 Keccak256(전역함수)와 같은 경우 바이트 레벨(Byte-level)'까지 신경 써야 함
```
bytes32 ers = keccak256(bytes("HackQuest"));

bytes memory data = "hello"
```
