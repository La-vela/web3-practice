### mapping add / update

매핑을 추가하고 업데이트하는 구문

```
mapping(address => uint256) public balance;

balance[address(0x123)] = 10; //this assigned a new value to address 0x123)
balance[address(0x123)] = 20; //this updated the value from 10 to 20
```


### contract


```
contract ContractName {
    //This is a state variable
    int a;

    function add(int b) return(int){
        //b is defined as a parameter, so it's not state variable
        //c is defined in a function, so it's also not a state variable
        int c = a + b;
        return c;
    }
}
```
