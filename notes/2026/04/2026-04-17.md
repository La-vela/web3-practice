## 개념

- 상속 (Inheritance): abstract contract Ownable은 직접 배포할 수 없고, 다른 컨트랙트(Vault 등)가 이를 상속(is Ownable)받아 사용해야 한다.
- Modifier(onlyOwner): 특정 함수를 주인만 실행하게 만드는 "자물쇠" 역할을 하는 기능, _(underscore)기호가 실제 함수의 코드가 실행되는 시점을 의미한다.
- Counstructor: 배포 시 initialOwner 주소를 전달해야 한다.

## 에러 해결

- Pragma 버전은 일치했으니 문제 없음
- 코드상 Import 경로 문제 확인 및 수정
- 생성자 (Constructor) 누락


## 코드
```
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

import " @openzeppelin/constracts/access/Ownable.sol";

constract Vault is Ownable {
    event Deposit(address indexed sender, uint256 amount);
    event Withdraw(uint256 amount);

    // 배포 시 나를 주인으로 설정
    constructor() Ownable(msg.sender) {}

    // 입금 기능
    receive() external payable {
        emit Deposit(msg.sender, msg.value);
    }

    // 출금 기능 (주인만 가능)
    function withdraw() external onlyOwner {
        uint256 balance = address(this).balance;
        payalber(owner()).transfer(balance);
        emit Withdraw(balance);
    }
}
```
